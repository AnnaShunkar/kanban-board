# Kan Ban board.
# Info:
A kanban board is a visual project management tool that helps teams manage workflow, limit work-in-progress, and maximize efficiency across projects.

Boards use cards, columns, and WIP limits to visualize tasks and identify bottlenecks in real time.

Kanban boards can be physical or digital, and are adaptable to various team processes and contexts.
